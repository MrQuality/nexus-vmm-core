# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of nexus-vmm-core seriously. If you discover a security vulnerability, please do NOT open a public issue. 

Instead, please report it privately by emailing: **mrnirrozen+nexus@gmail.com**.

We will acknowledge receipt of your vulnerability report as soon as possible and work with you to understand and remediate the issue.
