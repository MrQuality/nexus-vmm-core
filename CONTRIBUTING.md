# Contributing to nexus-vmm-core

First off, thank you for considering contributing to nexus-vmm-core!

## Developer Certificate of Origin (DCO)

To ensure that contributors have the right to submit code under our Apache 2.0 license, we enforce a Developer Certificate of Origin (DCO). 

All commits must be signed off to indicate that you agree to the terms in the `DCO` file. You can sign off your commits using the `-s` or `--signoff` flag:

```bash
git commit -s -m "feat: add awesome feature"
```

## Local Tests

Before submitting a pull request, please ensure that all local tests pass. 

## Code Review Standards

- Ensure your code follows the established formatting and style guidelines.
- Provide a clear and descriptive pull request title and description.
- Include new tests or update existing ones for any new features or bug fixes.
- Respond promptly to feedback during the code review process.
